markdown# 마크다운(Markdown) 문법 총정리

마크다운은 일반 텍스트 문서의 양식을 편집하는 경량 마크업 언어입니다.

---

## 1. 제목 (Headers)
`#` 개수로 글자 크기를 조절합니다. (1개부터 6개까지 가능)

# 제목 1 (H1)
## 제목 2 (H2)
### 제목 3 (H3)
#### 제목 4 (H4)
##### 제목 5 (H5)
###### 제목 6 (H6)

---

## 2. 텍스트 강조 (Emphasis)
글자를 굵게 하거나 기울일 때 사용합니다.

*기울임꼴* 또는 _기울임꼴_
**굵은 글씨** 또는 __굵은 글씨__
***굵은 기울임꼴***
~~취소선~~

---

## 3. 목록 (Lists)
순서가 있는 목록과 없는 목록을 만듭니다.

### 순서 없는 목록 (`-`, `*`, `+` 사용)
- 항목 1
- 항목 2
  - 하위 항목 (스페이스바 2칸 또는 Tab 누르고 작성)

### 순서 있는 목록 (숫자 사용)
1. 첫 번째
2. 두 번째
3. 세 번째

---

## 4. 링크와 이미지 (Links & Images)
웹 주소를 연결하거나 이미지를 불러옵니다.

[네이버 바로가기](https://naver.com)
![대체 텍스트(이미지가 깨졌을 때 출력)](https://placeholder.com)

---

## 5. 인용구 (Blockquotes)
문단 앞에 `>`를 붙여 강조된 인용 박스를 만듭니다.

> 이것은 인용구입니다.
>> 꺾쇠를 두 개 쓰면 중첩 인용구가 됩니다.

---

## 6. 코드 (Code)
백틱( ` ) 기호를 사용하여 코드 영역을 표시합니다.

### 인라인 코드
문장 중간에 `System.out.println();`처럼 코드를 넣을 때 씁니다.

### 코드 블록
백틱 3개를 사용하고 언어 이름을 적으면 문법 강조(Highlighting)가 적용됩니다.

```python
def welcome():
    print("Hello, Markdown!")
```

---

## 7. 가로 구분선 (Horizontal Rules)
하이픈(`-`) 또는 별표(`*`)를 3개 이상 연속으로 입력합니다.

---

---

## 8. 표 (Tables)
수직선(`|`)과 하이픈(`-`)으로 표를 그립니다. 콜론(`:`) 위치로 정렬을 결정합니다.

| 이름 | 나이 | 역할 |
| :--- | :---: | ---: |
| 홍길동 | 20 | 왼쪽 정렬 |
| 이순신 | 40 | 가운데 정렬 |
| 임꺽정 | 30 | 오른쪽 정렬 |
